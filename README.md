# Currency Converter — конвертер валют с графическим интерфейсом

**Автор:** Леонтьев Дмитрий Витальевич
**Дата:** 01.05.2026  

Приложение позволяет конвертировать суммы из одной валюты в другую по актуальному курсу с помощью внешнего API, сохраняет историю операций в JSON-файл и отображает её в удобной таблице. Проект прошёл полное тестирование (позитивные, негативные, граничные случаи) и готов к сдаче.

---

## 📥 Установка и запуск

### 1. Требования
- Python 3.7 или выше
- Библиотека `requests`

### 2. Получение API-ключа
Приложение использует бесплатный сервис [ExchangeRate API](https://www.exchangerate-api.com/).  
**Пошаговая инструкция:**
1. Перейдите на [страницу регистрации](https://app.exchangerate-api.com/sign-up).
2. Зарегистрируйтесь (через Google или e-mail).
3. После входа в панель управления скопируйте ваш **API-ключ** (начинается с `a1b2c3...`).
4. В файле `currency_converter.py` найдите строку:
   ```python
   API_KEY = "ваш_api_ключ"
и замените "ваш_api_ключ" на скопированный ключ.

Бесплатный тариф даёт 1500 запросов в месяц – достаточно для личного использования.

3. Установка зависимостей
bash
pip install requests
4. Запуск
bash
python currency_converter.py
🧑‍💻 Использование
Выбор валют – из выпадающих списков «Из валюты» и «В валюту».

Ввод суммы – положительное число (дробная часть через точку).

Конвертация – нажмите кнопку «Конвертировать».

Результат – появится под кнопкой, а запись добавится в таблицу истории.

Очистка истории – кнопка внизу окна (с подтверждением).

💾 Файл истории
Все операции сохраняются в файл history.json в той же папке, что и программа.
Формат записи:

json
[
  {
    "datetime": "2026-05-01 14:23:05",
    "from_currency": "USD",
    "from_amount": 100,
    "to_currency": "EUR",
    "to_amount": 92.45,
    "rate": 0.9245
  }
]
При каждом запуске история автоматически загружается из JSON. Кнопка «Очистить историю» удаляет все записи как из интерфейса, так и из файла.

🔧 Технические детали
API: https://api.exchangerate-api.com/v4/latest/USD (базовая валюта USD).

Конвертация: промежуточный пересчёт в USD → поддержка любых валютных пар.

GUI: tkinter (встроенная библиотека) + ttk.Treeview для таблицы.

Многопоточность: загрузка курсов в фоновом потоке — интерфейс не «зависает».

JSON: сохранение и загрузка через модуль json с кодировкой UTF-8.
